Stack (push): cada tela empilha sobre a anterior.
Voltar sempre disponível. Checklist é a tela central — scroll pelos itens.

A — Stack linear (recomendado)

Lista → Nova Auditoria → Checklist com todos os itens → Tela de item (ao tocar) → Resumo final. Fluxo direto, um caminho claro.

Tela de Detalhes tem tabs internas (Checklist / Resumo). Item abre como bottom sheet.

B — Detalhes com tabs internas

Lista → Detalhes (tab Checklist / tab Resumo) → Item como bottom sheet. Mais compacto, menos navegação, mas tabs adicionam complexidade.
